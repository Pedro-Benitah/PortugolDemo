# Demonstração Completa do Compilador Portugol

## 1. Visão Geral

Este projeto implementa um **compilador puro em C# .NET 8.0** para a linguagem Portugol (linguagem de programação educacional em português), com análise léxica e sintática baseadas em LL(1).

### Características Principais
- ✅ Análise léxica manual (sem ferramentas externas como Flex)
- ✅ Análise sintática LL(1) com stack-based parsing
- ✅ Suporte a 30+ tipos de tokens
- ✅ Gramática sem ambiguidades (eliminação de recursão à esquerda)
- ✅ Cálculo de conjuntos FIRST/FOLLOW
- ✅ Construção de tabela de parsing LL(1)
- ✅ Interface CLI com múltiplos modos de saída
- ✅ Exportação de tokens em CSV
